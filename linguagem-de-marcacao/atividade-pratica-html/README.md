# Atividade Prática HTML

Esta pasta reúne atividades práticas desenvolvidas em **HTML** com uso de **JavaScript** dentro dos próprios arquivos HTML.

As atividades trabalham conceitos iniciais de lógica de programação aplicados ao front-end, como constantes, variáveis, cálculos, condições e regras simples de sistema.

## Objetivo

O objetivo desta atividade é praticar a criação de páginas HTML com scripts em JavaScript, entendendo como a lógica pode ser aplicada dentro de uma página web.

## Atividades desenvolvidas

| Atividade | Tema | Conceitos trabalhados |
|---|---|---|
| Atividade 1 | Sistema de Média Escolar | Constantes, cálculo de média e regra de aprovação |
| Atividade 2 | Sistema de Velocidade de Jogo | Constantes e lógica aplicada a jogos |
| Atividade 3 | Sistema de Limite de Usuários | Regras de sistema e limite máximo |
| Atividade 4 | Sistema de Frete Grátis | Lógica de negócio e e-commerce |
| Atividade 5 | Sistema de Pontuação de Jogo | Pontuação e classificação |
| Atividade 6 | Sistema de Cadastro com Idade Mínima | Condições e validação de idade |
| Atividade 7 | Sistema de Compra com Desconto | Cálculo, desconto e lógica de compra |

## Conceitos estudados

- Estrutura básica de HTML
- Uso da tag `<script>`
- Constantes
- Variáveis
- Cálculos com JavaScript
- Condições
- Regras de sistema
- Exibição de informações no navegador ou console

## Como visualizar

Não é necessário instalar nenhuma dependência.

Para abrir as atividades:

1. Baixe ou clone o repositório.
2. Acesse a pasta da atividade desejada.
3. Abra o arquivo `.html` no navegador.
