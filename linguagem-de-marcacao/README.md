# Linguagem de Marcação

Este repositório reúne atividades, exemplos e projetos desenvolvidos durante os estudos de **Linguagem de Marcação**, com foco principalmente em **HTML** e estruturação de páginas web.

## Objetivo

O objetivo deste repositório é organizar os conteúdos estudados em aula, registrar a evolução dos exercícios e manter os projetos versionados de forma clara.

## Conteúdos abordados

- Estrutura básica de um documento HTML
- Tags semânticas
- Títulos, parágrafos e listas
- Links e imagens
- Tabelas
- Formulários
- Organização de páginas web
- Boas práticas de marcação
- Introdução à estrutura de sites
