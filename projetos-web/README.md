# Projetos Web

Esta pasta reúne projetos web desenvolvidos ao longo do curso Técnico em Desenvolvimento de Sistemas do SENAI-SP.

## Objetivo

Organizar projetos mais completos que envolvem estruturação de páginas, estilização, interatividade e construção de sistemas simples para a web.

## Conteúdos

Os projetos desta pasta podem envolver:

- HTML
- CSS
- JavaScript
- Interfaces web
- Sistemas escolares
- Páginas institucionais
- Projetos com múltiplos arquivos
- Organização de layout e navegação

## Finalidade

Registrar projetos práticos que demonstram a aplicação integrada dos conhecimentos desenvolvidos durante o curso.
