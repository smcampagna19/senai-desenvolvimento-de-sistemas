# Jogos

Esta pasta reúne jogos e atividades interativas desenvolvidas como prática de lógica de programação, interatividade e manipulação de elementos na tela.

## Objetivo

Organizar projetos que utilizam programação para criar desafios, interações, pontuação, movimentação ou mecânicas simples de jogo.

## Conteúdos

Os projetos desta pasta podem envolver:

- Lógica de programação
- JavaScript
- Condicionais
- Laços de repetição
- Eventos
- Interação com o usuário
- Manipulação de elementos da página

## Finalidade

Registrar atividades práticas que aplicam conceitos de programação em experiências interativas.
